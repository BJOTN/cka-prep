# List CRDs and Extract Field Documentation

**Task 1** — Create a list of all cert-manager CRDs and save it to `/root/resources.yaml`.

**Task 2** — Using `kubectl`, extract the documentation for the `subject` specification field of the `Certificate` Custom Resource and save it to `/root/subject.yaml`.

You may use any output format that `kubectl` supports.
