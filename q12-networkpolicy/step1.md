# Apply the Correct NetworkPolicy

There are two deployments — `frontend` in the `frontend` namespace and `backend` in the `backend` namespace.

**Task** — Look at the three NetworkPolicy YAML files in `/root/network-policies/`:
- `network-policy-1.yaml`
- `network-policy-2.yaml`
- `network-policy-3.yaml`

Decide which policy provides the functionality to allow interaction between the frontend and backend deployments in the **least permissive way**, then apply that file.
